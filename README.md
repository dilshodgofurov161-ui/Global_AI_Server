# 🚀 O'zingizning Shaxsiy Global AI Serveringiz

Tabriklayman! Endi sizda boshqalarning (OpenAI, Gemini) apilariga bog'lanmagan, 100% o'zingizning mustaqil AI serveringiz bor! U `node-nlp` orqali faqat o'z miyasidan foydalanib gaplashadi.

## Qanday qilib Global qilish mumkin? (Render.com ga bepul yuklash)

Sizda Node.js kompyuteringizda o'rnatilmagan bo'lsa ham, buni **tekinga butun dunyoga** onlayn qilish juda oson:

1. **GitHub** dan akkaunt oching va yangi "Repository" (loyihalar papkasi) yarating.
2. Shu papkadagi barcha fayllarni (`package.json`, `server.js`, `public` papkasi) GitHub ga yuklang.
3. **Render.com** saytiga kiring va GitHub orqali ro'yxatdan o'ting.
4. Render da **"New Web Service"** tugmasini bosing va o'zingizning GitHub dagi repo'ni tanlang.
5. Render avtomatik ravishda buni ko'radi:
   - Build command: `npm install`
   - Start command: `npm start`
6. Barchasi tekin ishlaydi! Render sizga `https://sizning-ai-nomingiz.onrender.com` degan maxsus manzil beradi.
7. Endi shu manzilni xohlagan do'stingizga yuboring va ular sizning shaxsiy AI ingiz bilan gaplashishadi!

Hamma kodlar tayyor. AI qanday javob berishini o'zgartirmoqchi bo'lsangiz `server.js` faylini ochib, o'zgartirishingiz mumkin.
